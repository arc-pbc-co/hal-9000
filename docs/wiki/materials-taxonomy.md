## Using the Taxonomy

### Querying Topics

```python
from hal9000.categorize.taxonomy import create_materials_science_taxonomy

taxonomy = create_materials_science_taxonomy()

# Get topic by slug
topic = taxonomy.get_topic("solid-state-batteries")
print(f"Name: {topic.name}")
print(f"Keywords: {topic.keywords}")

# Find topics by keyword
matches = taxonomy.find_topics_by_keyword("LLZO")
for match in matches:
    print(f"Match: {match.name}")
```

### Extending the Taxonomy

```yaml
# In config/materials_science_taxonomy.yaml

topics:
  - name: Energy Storage
    children:
      - name: Solid-State Batteries
        children:
          # Add new subtopic
          - name: Oxide Electrolytes
            slug: oxide-electrolytes
            keywords:
              - LLZO
              - garnet electrolyte
              - NASICON
```

## Classification Rules

1. **Primary Topics**: Confidence ≥ 0.6
2. **Secondary Topics**: Confidence ≥ 0.3
3. **Hierarchy**: Child topics automatically include parents
4. **Multi-label**: Documents can have multiple topics
